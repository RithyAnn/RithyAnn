Computer Aided Melanoma Diagnosis Using Interpretable Deep Model

This is my master's graduation project. It aims to design an interpretable model to detect melanoma. The model borrows the design ideas of autoencoder and prototype network. While the classification working, it realizes the self-explanation of the model independently of the results through feature reconstruction and distance calculation.
This is a brave exploration in solving the black box problem, a major difficulty facing the field of machine learning today.

The model contains two key parts: an autoencoder network and a prototype network.
The autoencoder is used for feature extraction and vector reconstruction. The prototype network is used for classification purposes.

The reconstructed images can be used as pixel-space representations of the feature vectors extracted from the individual inputs within the latent space. They act as visual cues that reflect the essential features extracted by the encoder, thus facilitating in-depth analysis of important features and the overall performance of the network. Through these reconstructed images, one can gain insight into the complexity of the feature extraction and reconstruction process, thus helping to understand the underlying mechanisms of the model's functionality.

A prototype image is a visual representation that captures the distribution and clustering of input features in pixel space, where each prototype image corresponds to a different class. These prototype images effectively showcase the features in samples that belong to a particular class. They provide an intuitive understanding of the unique characteristics of each class by visualizing the distribution and clustering of input features in pixel space. They play a vital role in example-based explanations, helping to explain classification decisions by comparing the similarities between input and prototype images.
