System And Equipment For Computer-Aided Retinopathy Detection

The System And Equipment For Computer-Aided Retinopathy Detection project started in 2019 and is carried out by the same team as the YUNYI GO (Go Referee System Based On Computer Vision And Machine Learning) project.
This project aims to provide efficient and low-cost testing methods for diabetic retinopathy in poor and medically backward areas.
We simultaneously developed a retinal image acquisition device and a retinal image analysis and processing network.
In this project, my main job is to participate in the construction of neural networks.
The project also received financial support from the China University Students’ Innovation and Entrepreneurship Program.
This project has achieved considerable results in The 5th China International College Students’ “Internet+” Innovation and Entrepreneurship Competition and the Challenge Cup.
